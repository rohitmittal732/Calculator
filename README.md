# Calculator
A simple Python Calculator that performs basic operations like addition, subtraction, multiplication, and division. Includes zero-division protection and a clean command-line interface. Perfect for beginners learning Python.
This is a simple Python-based Calculator application that performs basic arithmetic operations.
